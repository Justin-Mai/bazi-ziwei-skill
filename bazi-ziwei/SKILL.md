# 传统命理排盘分析技能 (八字与紫微斗数)

本技能通过底层的 Node.js 脚本，基于权威历法库 `lunar-javascript` 与紫微算法库 `iztro`，实现对生辰八字与紫微斗数星盘的精确计算。它能够结合经度与均时差进行真太阳时校准，支持早晚子时处理，并为命理分析、流年推算、大事定盘校准提供坚实的数据支撑。

## 1. 用户信息收集与对话确认规范

当用户提及“算命”、“排盘”、“批八字”、“看紫微斗数”等命理咨询请求时，**切勿直接盲目排盘或直接生成解盘分析**。你必须首先通过对话引导用户提供完整且准确的信息，并显式向用户确认是否进行真太阳时校准：

1. **引导收集基础信息**：
   询问并收集用户的：姓名（可选）、性别（男/女）、出生历法（公历/农历）、出生年月日以及**具体到分钟的出生时间**（如：09:09）。
2. **主动询问出生地点并确认真太阳时校准**：
   * 必须向用户解释并确认真太阳时校准：“为了确保时辰的绝对精准，排盘默认会启用**真太阳时校准**（结合您的出生地经度和地球公转时差，计算出真实的太阳出生时间）。如果您不想启用校准，请告知我。”
   * **时辰交接特别说明**：若用户的出生时间处于两个时辰的交界前后（例如：11:00、13:00 等整点前后，或子时跨天前后），你必须主动提醒用户：“**由于您的出生时间非常接近时辰交接点，是否使用真太阳时校准可能会产生两种完全不同的排盘结果（比如从巳时变为午时，或者时柱干支发生变化）。因此，请务必提供您准确的出生地点，我们会为您分析最精准的天地之气。**”
   * 询问用户的**出生具体地点（省市县，如‘广东湛江廉江’）**，以便通过经度数据库自动换算为真太阳时。
3. **确认后再执行脚本**：
   在获得上述明确的输入（尤其是确认了出生地以及用户对真太阳时校准的选择）后，方可进入下一步运行排盘脚本。

## 2. 核心工具调用

当上述基础信息与校准选项确认完毕后，**必须使用命令行运行排盘脚本**，以获取包含四柱干支、藏干、十神、纳音、五行力量、紫微十二宫星曜分布、大运及流年信息的 JSON 数据。

### 脚本路径
排盘脚本位于当前技能安装目录下的 `scripts/bazi_ziwei_cli.js`。

> [!IMPORTANT]
> 执行前，请先确定你当前加载的技能根目录（在不同 Agent 平台或用户机器上可能会有差异，例如在 Workbuddy 下为 `C:\Users\11215\.workbuddy\skills\bazi-ziwei\scripts\bazi_ziwei_cli.js`，在 Gemini/Antigravity 中为 `C:\Users\11215\.gemini\config\skills\bazi-ziwei\scripts\bazi_ziwei_cli.js`）。请务必替换为该脚本的实际绝对路径后再运行。

### 命令行参数
*   `--name`: 缘主姓名（默认“有缘人”）
*   `--gender`: 性别（“男”、“女”或“乾造”、“坤造”）
*   `--type`: 历法类型（`solar` 表示公历/阳历，`lunar` 表示农历/阴历。默认 `solar`）
*   `--date`: 出生日期（格式：`YYYY-MM-DD`，**必填**）
*   `--time`: 出生具体时间（格式：`HH:mm`，默认 `09:00`）
*   `--longitude`: 实测经度（东经°，可选）
*   `--location`: 出生省市县地点（可选，如 `"广东湛江廉江"`。若未传 `--longitude`，系统会自动通过内置的全国主要城市经度数据库查找对应的经度值并进行计算）
*   `--bazi-calib`: 是否启用八字真太阳时校准（`true` 或 `false`，默认 `true`）
*   `--ziwei-calib`: 是否启用紫微真太阳时校准（`true` 或 `false`，默认 `true`）
*   `--leap-month`: 农历类型时，是否是闰月出生（`true` 或 `false`，默认 `false`）

### 运行示例
```bash
# 请将 [bazi_ziwei_cli.js_path] 替换为该脚本在当前系统的实际绝对路径
node "[bazi_ziwei_cli.js_path]" --name "诸葛乾坤" --gender "男" --type "solar" --date "1995-10-10" --time "09:09" --location "广东湛江廉江" --bazi-calib true --ziwei-calib true
```

## 3. 排盘数据结构与分析指引

脚本输出的 JSON 包含三大核心板块：`meta`（元数据及真太阳时校准）、`bazi`（八字命盘与大运）、`ziwei`（紫微斗数十二宫星盘）。

### A. 真太阳时校准 (Meta Panel)
*   **经度时差 (longitudeDiffMinutes)**：根据当地经度与北京时间 120°E 的偏差计算（每度 4 分钟）。
*   **均时差 (eotMinutes)**：由于地球公转轨道为椭圆及黄赤交角，太阳每天在天球上的视运动速度不均匀导致的真时差。
*   **校准真太阳时 (trueTime)**：真实出生时间。此校准决定了八字的时柱干支是否会因为时区/公转时差而发生跨日、跨时辰变动，解决早晚子时及跨天排盘问题。
*   *分析时需向用户简要展示此校准过程，体现专业度。*

### B. 八字子平分析 (Bazi Panel)
1.  **日主旺衰与格局**：
    *   分析日干五行（如“甲木”）在月令（月支，如“酉月”为胎死之地或“寅月”为临官得禄）的得令、得地、得生帮情况。
    *   结合全局生扶克泄，判定身强或身弱。
    *   推断格局：如食神生财、官印相生、伤官配印等正格，或从儿、从财等变格。
2.  **五行力量与喜用忌神**：
    *   参考 JSON 中 `bazi.wuxing` 的五行力量量化百分比。
    *   找出全局最缺、最旺的五行，推算喜神、用神、忌神、仇神。
3.  **性格与命运特质**：
    *   结合十神（比劫、食伤、财星、官杀、印星）在四柱的透干与藏干分布。
4.  **六亲与婚姻**：
    *   日支为配偶宫，分析日支的十神及刑冲克合。
5.  **大运流年走势**：
    *   按起运年龄（如 6 岁起运）结合大运列表（`bazi.yun.list`），详细解析当下大运的吉凶，并指出近年流年互动的关口。

### C. 紫微斗数分析 (Ziwei Panel)
1.  **三方四正与命身宫**：
    *   命宫主星（如“紫微”、“武曲”）与命宫三方四正（财帛宫、官禄宫、迁移宫）的会照情况。
    *   命主（先天灵魂归宿）与身主（后天执着与中年后的发展重心）。
2.  **四化飞星**：
    *   **生年四化**：分析化禄（财源、缘分）、化权（权力、掌控）、化科（名声、科甲）、化忌（执念、欠债、阻碍）落在何宫，这是人生的核心动力和课题所在。
    *   结合流年命宫及流年四化，看当年的机遇。
3.  **宫位联动**：
    *   通过 JSON 中的 `ziwei.palaces` 字段，可读取十二宫各自的星曜分布（主星、辅星、煞星、杂曜、长生运等）。

*   详细解读日元（如甲木、癸水）的强弱及月令乘气。
*   判定具体命局格局（如正印格、食神制杀格）并解释成败。

### 2. 喜用神与忌神判定

### 3. 性格、事业与财运
*   分析求财路线，适合的职业方向（结合五行喜忌与十神分布）。

### 4. 感情婚姻与健康调理
*   解析配偶星与配偶宫的冲合，给出姻缘与身体健康建议。

## 5. 命理定盘 (时辰校准) 流程

若用户无法确定出生时辰，请遵循以下流程进行定盘：
1.  **收集备选盘**：让用户在不同备选时辰（例如申时、酉时）分别执行排盘脚本，并收集排盘数据。
2.  **索取大事线索**：让用户提供生平大事年份（如 2018 年考入重点大学、2021 年大病住院、2023 年结婚等）。
3.  **对比大限流年**：
    *   **对照八字**：分析各个备选时辰下，流年大运干支是否与原局发生重大刑冲克合（例如住院流年是否有羊刃逢冲或枭神夺食）。
    *   **对照紫微**：查看各盘中，大事年份的流年命宫、流年三方四正是否见四化（如结婚年是否流年夫妻宫见禄科、或有红鸾天喜会照）。
4.  **得出定盘结论**：给出各个盘口的吻合度打分（如盘一 90%，盘二 40%），说明支撑依据与不符合的矛盾点，最终确定最契合其生命轨迹的真实时辰。
