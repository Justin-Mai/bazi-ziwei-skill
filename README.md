# 🌟 玄机阁 · 传统命理排盘分析工具 (bazi-ziwei)

本工具是一个基于 **Node.js** 的传统命理排盘系统，融合了**八字子平术**与**紫微斗数**的排盘计算和深度命局解盘分析。通过结合天文历法库与真太阳时校准算法，为命理分析、大运流年推算以及大事定盘校准提供精确的数据支撑。

本工具不仅可作为独立的命令行排盘工具使用，同时也作为 AI 智能体的专属命理排盘技能（Agent Skill），赋能 AI 助手为用户撰写专业、深度、保真的命理解盘报告。

---

## 🚀 核心特性

- **⏱️ 真太阳时校准**：结合出生地的经度偏差与均时差（Equation of Time）进行真太阳时校准，避免因平太阳时（北京时间）导致的时辰甚至日期偏差。
- **🌍 智能地点定位**：内置全国主要省市经度数据库，支持通过输入出生地点（如“广东湛江廉江”）自动匹配经度并联动真太阳时计算。
- **🌓 早晚子时兼容**：完美支持早子时（00:00-01:00）与晚子时（23:00-24:00）的跨日排盘，符合子平命理传统。
- **🌌 独创八字五行力量加权**：基于地支藏干比例加权计算金、木、水、火、土五行力量的百分比，提供精准的喜用神判定依据。
- **🔮 紫微十二宫与生年/流年四化**：基于 `iztro` 算法库，生成紫微斗数十二宫星曜分布（主星、辅星、煞星、杂曜、长生运等），并计算生年四化、大限流年四化。
- **🔍 智能定盘辅助**：提供流年虚岁、小限虚岁与流年大限的映射数据，便于对生平大事进行时辰校准。

## 📦 安装与配置

### 前提条件
- 已安装 [Node.js](https://nodejs.org/) (建议 v14 或更高版本)

### 安装步骤

1. 克隆或下载本项目到本地。
2. 进入 `scripts` 目录并安装依赖包：
   ```bash
   cd scripts
   npm install
   ```

### 依赖包说明
本项目依赖于以下两个权威的排盘计算库：
- [lunar-javascript](https://github.com/6tail/lunar-javascript)：用于公农历转换、干支计算、八字排盘及均时差计算。
- [iztro](https://github.com/sinleree/iztro)：用于紫微斗数的星曜排布、三方四正与四化飞星计算。

## 💻 CLI 命令行使用指南

核心排盘逻辑通过 `scripts/bazi_ziwei_cli.js` 脚本运行，运行后会输出格式化后的排盘 JSON 数据。

### 1. 基础运行示例

**使用阳历（公历）出生日期排盘：**
```bash
node scripts/bazi_ziwei_cli.js --name "有缘人" --gender "男" --type "solar" --date "1995-10-10" --time "09:09"
```

**使用农历（阴历）出生日期排盘：**
```bash
node scripts/bazi_ziwei_cli.js --name "有缘人" --gender "女" --type "lunar" --date "1995-09-16" --time "09:09" --leap-month false
```

### 2. 真太阳时校准与地点联动示例

**根据具体出生经度进行精确校准：**
```bash
node scripts/bazi_ziwei_cli.js --name "诸葛乾坤" --gender "乾造" --date "1995-10-10" --time "09:09" --longitude 110.28
```

**输入出生地点（系统将自动查表获取经度）：**
```bash
node scripts/bazi_ziwei_cli.js --name "诸葛乾坤" --gender "乾造" --date "1995-10-10" --time "09:09" --location "广东湛江廉江"
```

### 3. 命令行参数详解

| 参数名 | 类型 | 是否必填 | 默认值 | 参数说明 |
| :--- | :--- | :--- | :--- | :--- |
| `--date` | String | **是** | - | 出生日期，格式为 `YYYY-MM-DD` |
| `--time` | String | 否 | `09:00` | 出生具体时间，格式为 `HH:mm` |
| `--name` | String | 否 | `有缘人` | 缘主姓名 |
| `--gender` | String | 否 | `男` | 性别，支持 `男`/`女` 或 `乾造`/`坤造` |
| `--type` | String | 否 | `solar` | 历法类型，`solar` 表示公历/阳历，`lunar` 表示农历/阴历 |
| `--location` | String | 否 | - | 出生地省市县名称（如 `"广东湛江廉江"`），用于自动转换经度 |
| `--longitude`| Number | 否 | `120.0` | 确切出生经度（若同时传入 location，以此经度优先） |
| `--bazi-calib`| Boolean| 否 | `true` | 是否启用八字真太阳时校准 (`true`/`false`) |
| `--ziwei-calib`| Boolean| 否 | `true` | 是否启用紫微真太阳时校准 (`true`/`false`) |
| `--leap-month`| Boolean| 否 | `false` | 若 `--type` 为 `lunar`，是否为闰月出生 (`true`/`false`) |

## 📝 命理解盘报告撰写规范

为了更好地让 AI 智能体或分析系统生成深度、充实且有洞察力的“终身命理深度批命书”，建议解盘报告包含以下五个核心板块（总篇幅建议 1500 字以上）：

1. **基本盘口展示**：包含出生时间（公历/农历）、性别、以及真太阳时校准的过程（经度时差与均时差）。
2. **子平八字深度命局分析**：
   - **日主天干与格局推断**：通过日干五行在月令的衰旺状态，确定身强身弱，判定格局（如食神生财、正官配印等）。
   - **喜用忌神判定**：结合五行力量加权百分比，精准列出喜神、用神、忌神、仇神。
   - **性格、事业与财运**：分析求财路线，指出最适合的行业方位与职业建议。
   - **姻缘与健康**：通过配偶星与配偶宫的冲合克泄分析姻缘，并根据五行偏枯情况给出健康调理建议。
3. **紫微斗数命局精析**：
   - **三方四正与命身宫**：分析命宫主星及财帛、官禄、迁移宫的星曜组合。
   - **核心宫位联动与生年四化**：详解禄、权、科、忌的落宫，指明一生执念、障碍与福气所在。
4. **八字与紫微命理合参（信息共振）**：
   - 提取八字与紫微命盘中互相印证的信息（如八字喜木火，紫微命宫适逢太阳化禄，产生强烈的信息共振，增强推论可信度）。
   - 释疑两套体系的表面冲突（如八字财星受克，而紫微财帛宫吉星云集，应解析为“虽进财有波折，但终能大发”）。
5. **十年大运与流年运势推演**：
   - 详解当前大运（大限）的吉凶主旋律。
   - 推演今年及近年流年的运势变化与重要关口。
6. **改运与修心具体建议**：
   - 结合性格缺陷给出性情修持方案。
   - 给出适合的开运色、方位、数字及行业作息调理。

## 🔍 命理定盘（时辰校准）流程

若遇到缘主出生时辰不准（例如“生于傍晚”、“不知道具体几点”），系统与 AI 助手可遵循以下步骤定盘：

1. **收集备选盘**：让用户在不同备选时辰（如申时、酉时）分别执行排盘脚本，并收集排盘数据。
2. **收集大事线索**：让用户提供其生平的大事发生年份（例如：2018 年考学、2021 年生病住院、2023 年结婚等）。
3. **对照验证**：
   - **八字对照**：分析各备选时辰原局在大运流年中的刑冲克合（如住院年份是否有羊刃逢冲或枭神夺食）。
   - **紫微对照**：查看各盘中，大事年份的流年命宫、流年三方四正是否见四化（如结婚年是否流年夫妻宫见禄科、或有红鸾天喜会照）。
4. **定盘结论**：通过比对吻合度，给出各个盘口的吻合度打分（如“申时盘吻合度 90%，酉时盘吻合度 40%”），说明依据，最终确定最契合生命轨迹的真实出生时辰。
